hSEO Plugin for Hotaru CMS
---------------------------------
Created by: Andreas Votteler
Author URL: http://www.trendkraft.de


Description:
-----------
swiss knife for hotaru seo. with innovations from germany.
this plugin seo optimized certain settings. functions available from the features list found.

Features:
-----------
- Automated generation: Title, Description, Canonical, Robots
- leverage browser caching
- avoidance of double content in friendly url
- open graph implementation

Instructions
------------
1. Upload the "hSEO" folder to your plugins folder.
2. Install it from Plugin Management in Admin.
3. Go to Admin -> Plugin Settings -> hSEO and edit the Settings (optional)
